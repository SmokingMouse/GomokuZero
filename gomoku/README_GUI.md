# Gomoku AI Battle Arena

一个功能完整的五子棋对战界面，支持多种对战模式。

## 功能特性

### 🎮 对战模式
- **人对AI**: 人类玩家 vs ZeroMCTS AI
- **AI对AI**: ZeroMCTS vs 纯MCTS，可以设置不同迭代次数
- **人对人**: 纯人类对战模式

### ⚙️ 可调参数
- **AI迭代次数**: 100-2000次（默认800次）
- **AI温度**: 0.0-1.0（控制探索性）
- **人类玩家颜色**: 可选择执黑（先手）或执白（后手）

### 🎯 界面功能
- **实时状态显示**: 当前玩家、游戏模式、AI思考状态
- **控制面板**: 右侧显示游戏信息和操作说明
- **设置对话框**: 图形化调整AI参数

## 使用方法

### 启动界面
```bash
python run_gui.py
```

### 操作说明
- **鼠标点击**: 在棋盘上点击下棋
- **空格键**: 切换游戏模式（人对AI → AI对AI → 人对人）
- **R键**: 重新开始当前对局
- **S键**: 打开设置对话框

### 界面布局
- **左侧**: 9×9围棋棋盘
- **右侧**: 控制面板，显示游戏状态、参数和操作说明

## AI算法说明

### ZeroMCTS (神经网络增强)
- 使用训练好的策略网络指导搜索
- 支持PUCT算法和Dirichlet噪声
- 默认使用 `models/gomoku_zero_9_plus_pro/policy_step_199500.pth` 模型

### 纯MCTS
- 传统蒙特卡罗树搜索
- 使用随机策略进行模拟
- 基于UCT算法选择动作

## 技术实现

### 依赖库
- `pygame`: 图形界面
- `tkinter`: 设置对话框
- `torch`: 神经网络推理
- `threading`: AI异步思考

### 文件结构
```
gui_new.py          # 主界面代码
run_gui.py          # 启动脚本
mcts.py            # 纯MCTS实现
zero_mcts.py       # ZeroMCTS实现
gomoku_env.py      # 游戏环境
policy.py          # 神经网络策略
```

## 注意事项

1. **模型文件**: 确保 `models/gomoku_zero_9_plus_pro/policy_step_199500.pth` 存在
2. **性能**: AI迭代次数越高，AI越强但思考时间越长
3. **线程安全**: AI思考在后台线程进行，不会阻塞界面
